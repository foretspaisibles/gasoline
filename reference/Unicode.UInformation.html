Classification predicates

This is a simplified interface to primary_category and general_category.

val is_letter : Unicode.uchar -> bool
The is_letter functions tests for a letter, that is, a character whose primary category is LETTER.
val is_alpha : Unicode.uchar -> bool
The is_alpha function tests for an alphabetic character. It is the same as is_letter.
val is_number : Unicode.uchar -> bool
The is_number function tests for a digit character, that is, a character whose primary category is NUMBER. Note that this definition do not match the definition of the standard isnumber function that matches only decimal digits.
val is_mark : Unicode.uchar -> bool
The is_mark functions tests for a mark, that is, a character whose primary category is MARK.
val is_separator : Unicode.uchar -> bool
The is_separator functions tests for a separator, that is, a character whose primary category is SEPARATOR.
val is_other : Unicode.uchar -> bool
The is_other functions tests for a other, that is, a character whose primary category is OTHER.
val is_symbol : Unicode.uchar -> bool
The is_symbol functions tests for a symbol, that is, a character whose primary category is SYMBOL.
val is_digit : Unicode.uchar -> bool
The is_digit function tests for a decimal digit.
val is_alnum : Unicode.uchar -> bool
The is_alnum function tests for an alphanumeric character, that is, either a letter or a number.
val is_ascii : Unicode.uchar -> bool
The is_ascii function tests for an ASCII character.
val is_latin : Unicode.uchar -> bool
The is_latin function tests for a latin character.
val is_cntrl : Unicode.uchar -> bool
The is_cntrl function tests for a control character.
val is_graph : Unicode.uchar -> bool
The is_graph function tests for a printing character. White space is not considered as a print character.
val is_lower : Unicode.uchar -> bool
The is_lower function tests for a lowercase character.
val is_upper : Unicode.uchar -> bool
The is_upper function tests for an uppercase character.
val is_title : Unicode.uchar -> bool
The is_title function tests for an titlecase character.
val is_print : Unicode.uchar -> bool
The is_print function tests for a printing character. White space is considered as a printing character.
val is_punctuation : Unicode.uchar -> bool
The is_punctuation function tests for a punctuation character.
val is_space : Unicode.uchar -> bool
The is_space function tests for a white-space character. These white space characters include, but do not limit to, ``\t'' ``\n'' ``\v'' ``\f'' ``\r'' `` ''.
val is_xdigit : Unicode.uchar -> bool
The is_xdigit function tests for a hexadecimal digit.
val is_odigit : Unicode.uchar -> bool
The is_odigit function tests for an octal digit.
val is_bdigit : Unicode.uchar -> bool
The is_bdigit function tests for a binary digit, that is, either ``0'' or ``1''.
val is_newline : Unicode.uchar -> bool
The is_newline function tests for a newline character.
