Module type Generic_type.CLASSIFICATION

module type CLASSIFICATION = sig .. end
Classifying messages.

A well known classification is the one used by the UNIX syslog facility. Parametrisation of this module allows to define new classification schemes, however, it is probably a good idea to use only classification consisting of a finite, fixed, hierarchised set, of tags.
