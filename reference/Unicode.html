Module Unicode

module Unicode: sig .. end
Unicode facility.

Camomile is a complete Unicode library, by Yamagata Yoriyuki. This Unicode module is a simple wrapper around some Camomile modules.

The recommanded use for this module is to be opened with the open Unicode statement.
