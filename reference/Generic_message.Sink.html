Module Generic_message.Sink

module Sink: sig .. end
Message sinks.

This interface is very similar to Sender but distinct and incompatible: the way values are created is different.

The main differences are the ability to defer connection to a message databank, and to attach several output channels to a message sink.

It is thus possible to use message sinks at the initialisation time of a program, delaying the databank connection to a later time.
