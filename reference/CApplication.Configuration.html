val make : 'a kind ->
       component ->
       ?flag:char ->
       ?env:string ->
       ?shy:bool -> string -> 'a -> string -> 'a t
make kind comp name default description create a configuration item holding a value of type 'a. This configuration value can be manipulated through the functions set and get below.
flag : the letter used for command line flag.
env : the environment variable used to get a value.
shy : flag governing description in the short help.
val get : 'a t -> 'a
get item return the value of the given configuration item.
val set : 'a t -> 'a -> unit
set item value set the value of the given configuration item.
