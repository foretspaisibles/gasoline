Generic application components.
type info = {
   name : string;
   version : string;
   description : string;
   require : string list;
   provide : string list;
   config_prefix : string list;
   getopt_prefix : char option;
}
The type of software component information cards.
type callback = info -> unit
The type of component callbacks.
type t
The abstract type of components.
val make : info ->
       callback ->
       callback -> t
make info boostrap shutdown prepares a software component identified by info and using the callbacks boostrap and shutdown.
val bootstrap : t list -> unit
Sorts the given component list according to the require and provide fields and bootstrap these in order.
Raises Failwith if the specification cannot be satisfied.
val shutdown : t list -> unit
Sorts the given component list according to the require and provide fields and shutdown these in order.
Raises Failwith if the specification cannot be satisfied.
