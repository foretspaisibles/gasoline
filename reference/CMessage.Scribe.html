Module CMessage.Scribe

module Scribe: Generic_message.Scribe.S 
  with type classification = Classification.t
   and type locale = Locale.t
   and type connection_token = Database.connection_token
   and type out_channel = Data.out_channel
   and type value = Value.t
   and type buffer = Buffer.t
   and type block = Data.block
Concrete C-like message facility.
type t
The type of message scribes.
type classification
The type of message classifications.
type connection_token
The type of message database connection tokens.
type out_channel
The type of output channels.
type value
The type of variable values that can occur in a message.
type locale
The type of locales.
type buffer
The type of data buffers.
type block
The type of data blocks.
val make : connection_token ->
       locale -> t
Make a message scribe.

The message scribe holds a handler to the database open thanks to the connection_token.

val add : t ->
       classification ->
       string -> (string * value) list -> unit
add scribe c id bindings add the characters of the message id classified as c and with bindings bindings to an internal buffer in scribe.
val clear : t -> unit
Empty internal buffer.
val reset : t -> unit
Empty and reset internal buffer.
val contents : t -> block
Return the contents of the internal buffer. The internal buffer itself is not changed.
val output : out_channel -> t -> unit
Output the contents of the internal buffer. The internal buffer itself is not changed.
val transfer : buffer -> t -> unit
transfer b scribe copy each word in the internal storage of scribe to b. The internal buffer of scribe is then emptied.
