run name usage description main run the daemon.

If the flag "-F" is given, then the daemon stays in the foreground. If stdout and or stderr are given and start with a character '/' then they are interpreted as files where stdin and stderr are redirected to. Otherwise, they are intpreted as subprocesses where these descriptors should be redirected to.
