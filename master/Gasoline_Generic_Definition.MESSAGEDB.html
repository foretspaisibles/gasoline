Module type Gasoline_Generic_Definition.MESSAGEDB

module type MESSAGEDB = sig .. end
Reading the message database.
type t
The abstract type of message database handles.
type connection_token
The abstract type of connection tokens.
val opendb : connection_token ->
       t
Open a descriptor on a message database.
Raises Failure when connection fails.
val find : t -> string -> string
find db key return a concrete form for the lexemes associated with id in the database opened for descriptor db.
Raises Not_found if the key has no associated data.
val close : t -> unit
Close the given descriptor.

Using a closed descriptor is an error and raises a Failure.
