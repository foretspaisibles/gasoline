val make : connection_token ->
       locale ->
       t
Make a message scribe.

The message scribe holds a handler to the database open thanks to the connection_token.

val add : t ->
       priority ->
       string -> (string * value) list -> unit
add scribe c id bindings add the characters of the message id classified as c and with bindings bindings to an internal buffer in scribe.
val clear : t -> unit
Empty internal buffer.
val reset : t -> unit
Empty and reset internal buffer.
val contents : t ->
       block
Return the contents of the internal buffer. The internal buffer itself is not changed.
val output : out_channel ->
       t -> unit
Output the contents of the internal buffer. The internal buffer itself is not changed.
val transfer : buffer ->
       t -> unit
transfer b scribe copy each word in the internal storage of scribe to b. The internal buffer of scribe is then emptied.
