Maps over Unicode characters

The type of the map keys.
type 'a t
The type of maps from type uchar to type 'a.
val empty : 'a t
The empty map.
val is_empty : 'a t -> bool
Test whether a map is empty or not.
val add : Gasoline_Unicode.uchar ->
       'a -> 'a t -> 'a t
add u v m returns a map containing the same bindings as m, plus a binding of u to u. If u was already bound in m, its previous binding disappears.
val add_range : Gasoline_Unicode.uchar ->
       Gasoline_Unicode.uchar ->
       'a -> 'a t -> 'a t
add u1 u2 v m returns the new map which is same to m except it maps characters in the range u1 - u2 to v.
val find : Gasoline_Unicode.uchar -> 'a t -> 'a
find u m returns the current binding of u in m, or raises Not_found if no such binding exists.
val remove : Gasoline_Unicode.uchar ->
       'a t -> 'a t
remove u m returns a map containing the same bindings as m, except for u which is unbound in the returned map.
val remove_range : Gasoline_Unicode.uchar ->
       Gasoline_Unicode.uchar ->
       'a t -> 'a t
remove_range u1 u2 m removes u1 - u2 from the domain of m
val from : Gasoline_Unicode.uchar ->
       'a t -> 'a t
from u m restricts the domain of m to the characters whose code points are equal or greater than u.
val after : Gasoline_Unicode.uchar ->
       'a t -> 'a t
after u m restricts the domain of m to the characters whose code points are greater than u.
val until : Gasoline_Unicode.uchar ->
       'a t -> 'a t
until u m restricts the domain of m to the characters whose code points are equal or smaller than u.
val before : Gasoline_Unicode.uchar ->
       'a t -> 'a t
before u m restricts the domain of m to the characters whose code points are smaller than u.
val mem : Gasoline_Unicode.uchar -> 'a t -> bool
mem u m returns true if m contains a binding for u, and false otherwise.
val iter : (Gasoline_Unicode.uchar -> 'a -> unit) -> 'a t -> unit
iter f m applies f to all bindings in map m. f receives the key as first argument, and the associated value as second argument. The bindings are passed to f in increasing order with respect to the ordering over the type of the keys. Only current bindings are presented to f: bindings hidden by more recent bindings are not passed to f.
val iter_range : (Gasoline_Unicode.uchar -> Gasoline_Unicode.uchar -> 'a -> unit) ->
       'a t -> unit
iter proc m : For each contingent region u1-u2 that is mapped to a constant v, proc u1 u2 v is called. The order of call is determined by increasing order on u1.
val map : ('a -> 'b) -> 'a t -> 'b t
map f m returns a map with same domain as m, where the associated value a of all bindings of m has been replaced by the result of the application of f to a. The bindings are passed to f in increasing order with respect to the ordering over the type of the keys.
val mapi : (Gasoline_Unicode.uchar -> 'a -> 'b) ->
       'a t -> 'b t
Same as UMap.S.map, but the function receives as arguments both the key and the associated value for each binding of the map.
val fold : (Gasoline_Unicode.uchar -> 'b -> 'a -> 'a) ->
       'b t -> 'a -> 'a
fold f m a computes (f kN dN ... (f k1 d1 a)...), where k1 ... kN are the keys of all bindings in m (in increasing order), and d1 ... dN are the associated data.
val fold_range : (Gasoline_Unicode.uchar -> Gasoline_Unicode.uchar -> 'b -> 'a -> 'a) ->
       'b t -> 'a -> 'a
fold_range f m x is equivalent to f u_(2n) u_(2n+1) v_n (... (f u_1 u_2 v_1 x)) where all characters in the range u_(2k)-u_(2k+1) are mapped to v_k and u_1 < u_3 < ... in code point order. For each range u_(2k)-u_(2k+1) is separated by a character which is not mapped to v_k.
val constant : Gasoline_Unicode.USet.t -> 'a -> 'a t
Constant map.
val domain : 'a t -> Gasoline_Unicode.USet.t
Domain.
val subdomain : ('a -> bool) -> 'a t -> Gasoline_Unicode.USet.t
subdomain p m returns the set of characters which are mapped to values satisfying the predicate p by m.
val compare : ('a -> 'a -> int) ->
       'a t -> 'a t -> int
Total ordering between maps. The first argument is a total ordering used to compare data associated with equal keys in the two maps.
val equal : ('a -> 'a -> bool) ->
       'a t -> 'a t -> bool
equal cmp m1 m2 tests whether the maps m1 and m2 are equal, that is, contain equal keys and associate them with equal data. cmp is the equality predicate used to compare the data associated with the keys.
