Module type Gasoline_Generic_Definition.LOCALE

module type LOCALE = sig .. end
Localising preparation of messages.

When preparing a value for its printing in a message, some decisions are tied to to the localisation parameters defined by the program operator, some others to the value itself. These are locale and format.

type t
The type of localisation parameters.
val stdlocale : t
The standard localisation parameters, analogous to the "C" locale for C-stylish programs.
