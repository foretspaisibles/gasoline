Module Gasoline_Plain_Definition.Priority

module Priority: sig .. end
type t =
| Debug
| Info
| Notice
| Warning
| Error
| Critical
| Alert (*
A condition that should be corrected immediately.
*)
| Emergency (*
A panic condition.
*)
We imitate the classification of the syslog facility. The normal classification for a message is Info. The highest Emergency, Alert or Critical error levels are most probably only relevant for critical systems. The highest level a typical command line utility should use is Error, usually indicating an unrecoverable error.
